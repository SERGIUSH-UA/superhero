# Test task Superheroes DB

Nestjs web application. It`s a server side part

## How to run project


## Installation

```bash
$ npm install
```
Reconfigure `.development.env` and `.production.env` file for you configuration

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
